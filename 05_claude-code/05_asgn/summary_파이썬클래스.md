# 파이썬 클래스 학습 정리

## 핵심 개념

| 요소 | 설명 |
|------|------|
| 클래스(Class) | 데이터와 기능을 묶는 설계도 |
| 객체(Instance) | 설계도를 바탕으로 만들어진 실체 |
| `__init__` | 객체 생성 시 자동 실행되는 초기화 메서드 |
| `self` | 만들어지는 객체 자신을 가리키는 참조 |

## 주요 내용

```python
class Dog:
    def __init__(self, name, age):  # 객체 생성 시 자동 실행
        self.name = name            # 이 객체에 name을 저장
        self.age = age

    def bark(self):
        print(f"{self.name}: 멍멍!")

dog1 = Dog("초코", 3)  # self = dog1
dog2 = Dog("콩이", 5)  # self = dog2

dog1.bark()  # 초코: 멍멍!
dog2.bark()  # 콩이: 멍멍!
```

- 같은 클래스로 만든 객체라도 각자의 데이터를 독립적으로 저장한다.
- `self.name = name` 은 해당 객체에 데이터를 **붙여두는** 것 → 함수 종료 후에도 유지된다.
- `self` 없이 `name = name` 만 쓰면 함수 내 임시 변수로 사라진다.

## 기억할 포인트

- 클래스는 틀, 객체는 틀로 찍어낸 결과물
- `self`는 "지금 만들어지고 있는 이 객체"를 가리킨다
- `__init__`은 객체를 초기화하는 생성자로, 객체 생성 시 자동 호출된다
- 각 객체는 독립된 메모리 공간을 가지므로 서로 다른 값을 저장할 수 있다
