# ollama-namu-agent 실습 결과 분석
예시 시나리오:
```bash
사용자: "젤다에 대해 검색해줘"
→ 에이전트가 자동으로: (1) 나무위키에서 "젤다" 검색 → (2) 검색 결과를 파일에 저장
```

## 실행 결과
- 에이전트 실행 결과 스크린샷 (검색 → 파일 저장 과정이 보이도록)
